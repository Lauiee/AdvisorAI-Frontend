# AdvisorAI-Frontend

AI 기반 Human Digital Twin으로 학생과 교수의 '최적의 인연'을 연결하는 매칭 플랫폼

## 기술 스택

- React 18
- Vite 5
- CSS3

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 브랜드 컬러

- Primary Color: `#1E2A3A` (Deep Blue)
- Secondary Color: `#32C3B0` (Teal/Mint)
- Neutral Background: `#F5F6FA` (Light Gray)
